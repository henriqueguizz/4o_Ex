# 4o_Ex

## Descrição 
No presente repositório se encontram os arquivos elaborados no 4º exercício de Ciência de Dados do curso de verão. São eles:
1) Código em R que almeja responder à pergunta "Qual a porcentagem de cobertura vegetal média de cada município do estado do Rio de Janeiro?"; e
2) Mapa elaborado ao longo do código, mostrando a porcentagem de cobertura vegetal nesses mesmos distritos.
